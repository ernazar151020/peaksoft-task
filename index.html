QUESTION :
Write a function to find the longest common prefix string amongst an array of strings.

If there is no common prefix, return an empty string "".

EXPLANATION :
Example 1 :
Input: strs = ["flower","flow","flight"]
Output: "fl"
Example 2 :
Input : strs = ["dog","racecar","car"]
Output: ""
Explanation : There is no common prefix among the input strings.

SOLUTION

1. WE CHECK LENGTH OF ARRAY . IF ARRAY IS EMPTY THAN RETURN "" EMPTY VALUE
2. LETS DECLARE 2 VARIABLES . let longest="" , let firstword=strs[0]
3.NOW WE ARE ON MOST COMPLEX PART . FIRST THING WE SHOULD DO IS ITERATE OVER let firstword=strs[0] WHY WE ITERATING OVER THIS VARIABLE ? ANSWER IS WE WILL compare the char of the first word with char of each of the other words IF THE CHARS ARE NOT MATCHING RETURN let longest="" THIS VARIABLE . FINALLY If all words have a matching char, append it to the longest common prefix.
